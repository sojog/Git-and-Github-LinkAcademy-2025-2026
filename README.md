# Git-and-Github-LinkAcademy-2025-2026

##  Comenzi principale



# Initializarea unui proiect git
git init        //  Acesta se executa o singura data




# Adaugarea unui fisier in staged 
git add nume_fisier        // adaugam un singur fisier sau toate cu "."
 
# Creare unui commit 
git commit -m " .... mesajul propriu zis ...  "

# Sincronizarea cu serverul
git push



## Comenzi vizualizare 

# Pentru a vedea toate commit-urile
git log 

# Pentru a vedea starea fisierelor
git status
